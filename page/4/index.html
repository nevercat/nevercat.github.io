盛天网络的那些事

唧唧歪歪生活些事 
    一个很意外的机会，决定在8月份放弃去第三课堂转而去企业实习，之后就收到公司的面试的通知，再然后稀里糊涂的就被录用了。

    今天是去湖北盛天网络技术股份有限公司实习的第一天，整体感觉还不错，感觉这家公司很有活力，一大群朝气蓬勃的年轻人在一块奋斗的感觉，企业文化很有特色。曾经一直以为挨踢程序猿都是苦逼的闷骚男，今天才发现我错了，这个公司的整个氛围和文化都很有活力和朝气，还很自由，朝九晚六，就上下班需要按指纹打一下卡，应该是记考勤吧。说实话我很喜欢这儿的工作环境，哈哈~尤其还是研发部单独给我们安排了个工作室，并且还配置了超高性能的电脑，当时看到电脑的配置我就吓到了，Intel Core i5-2320 @ 3.00GHz 四核处理器，金士顿8 GB内存，西数1 TB / 7200 转/分硬盘，Nvidia GeForce GT 610 ( 1024 MB / 七彩虹 )显卡，这不知道是我本子配置的多少倍，光内存就秒杀了，还1T硬盘，还是7200转的！实际操作看来非常流畅，丝毫感觉不到卡顿，网速也很给力哈~

    我是在研发部做数据抓包的工作，之前没来的时候觉得很困难，其实呢，是个很简单的工作，甚至有点枯燥，不过要做好还是有一定难度的。简而言之就是用WireShark或者WSexplorer之类的软件在网络传输过程中进行抓包，然后分析数据包的内容，基本不涉及编程吧，侧重网络方面。下学期就要开始上网络相关的课程了，想想现在提前了解下学习下也不错哈~然后今天公司的HR跟我们说在研发部还有搞php开发的一条线，我当时就挺开心的，正好这是个机会啊，我还可以利用多的时间去向他们请教php方面的问题呢……

    因为研发部现在正在进行的项目的特殊性，我们是每周二到每周六上班，周日和周一休息，所以本来明天是要上班的，但我之前就打算去南昌玩玩，并且票都买好了，所以只好请假了，其实上班第二天就请假感觉怪不好意思的，可HR和带我们的负责人都很亲切，批准了我的请假，很感谢哈。。

    so，祝我明天旅途愉快。。开开心心的回来后再继续fighting啦~

浏览器判断 - 令人头疼纠结的浏览器兼容问题

study
javascript

今天在人人上面看到华中科技大学思存工作室推出了一个2012年新生专题，一打开网页，在页首就出现个橙色的亮条，提示我“您正在使用IE6或IE7内核的浏览器，为了您更高的浏览体验，推荐您使用chrome/FireFox/Opara/IE8/IE9等浏览器来浏览本网站！”原来我用的是搜狗浏览器兼容模式，切换到高速模式后，亮条消失。IE内核的浏览器越来越不给力，尤其是IE6和IE7，IE8和IE9稍微好点但也越来越力不从心了，于是结合前几天处理自己博客浏览器兼容时遇到的问题，想到模仿这个用javascript写一个浏览器判断，也好用在自己的博客上，提示那些执着于IE6/7的童鞋们该升级浏览器啦~

百无聊赖的日子

唧唧歪歪生活些事 
    数数看回到家已经13天了，可是貌似什么都没做，天天在家不是睡觉就是吃饭，现在开始后悔了好好的湖北日报的网站前台制作实习没敢去。暑假已经过去三分之一，经过了前几天的暴雨，现在又开始燥热了。

    回来只带了一本书，《java编程思想》，够厚的一本神书，当然我看的是中文版，如果看英文版估计早就吐血了，刚回家时还有点兴致，看了几天就看不下去了，我总是没有这个恒心。不过现在呢还是在慢慢啃，包括一边在电脑上敲代码，但又一边漫不经心的走神跑去做其他的事了，比如说又去搞这个博客，换了个小清新的主题，进行了一些二次开发，然后在浏览器兼容问题上面伤透了脑筋，到现在，用IE6/7访问，首页右边的侧边栏应该会错位，但用WebKit内核的谷歌浏览器或者搜狗浏览器的高速模式访问就应该没问题，我下了各种浏览器来检查兼容却发现还是搞不好，IE内核的确对新的css解析很有问题，头痛死了。算了，不管了，友情劝告还在使用IE6/7的朋友们，你们该换个新的浏览器了！

    从5月份开始喜欢上了玩微博，然后也开始翻墙搞facebook和twitter，基本把人人和QQ空间抛弃了，一方面是发现爸妈开始关注我空间更新了，囧，不想让他们知道我每天的动态而不得不转移阵地，然后另一个很大的原因是很喜欢微博的开放性。昨天到今天在新浪的开放平台上申请API接口，想把自己的这个博客网站跟微博连起来，因为没有ICP备案，我就类型选的是境外网站，然后把买域名时的那个证书截图随便上传上去了，居然备案这一块就放我过了，@Acshiryu，你就是选国内网站没备案被驳回了吧，哈哈。。然后申请了2次，都被拒绝，第二次还被执行提交审核功能冻结，要到7月28号解封，本来想放弃的，今天上午突然想到用子域名重新申请个app key重新提交，结果竟然就过了，人品大爆发啊。现在通过网站发微博会显示来自简笔画，哇哈哈，小小的得瑟一下……

    心想着还是要在家做点什么好，下个月10号就要到学校去参加传说中的第三课堂，在家也就不到20天了，嗯嗯，加油咯~
